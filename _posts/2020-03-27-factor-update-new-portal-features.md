# <span style="color: #5e6970;">Factor Update: New Portal Features</span> {.h1-case-study-title}

\[/vc\_column\_text\]\[vc\_empty\_space height=&#8221;30px&#8221;\][vc\_column\_text]Hopefully you’re all staying healthy. This note is to let you know that we’ve added two new features to our Factoring platform:

&nbsp;

The ability to require specific fields in our client portal before clients submit invoices.

&nbsp;

From the portal, giving clients the ability to enter funding methods and automatically passing these into FactorSoft.

&nbsp;

HubTran uses these fields and funding information to automatically generate the client’s funding form.

&nbsp;

**1. Requiring specific fields from clients in portal**

By taking advantage of this feature, you’ll be able to make sure that you’re receiving complete submissions from your clients through our portal. To set which fields you want to require from your clients, start by navigating to the Settings tab on the top navigation bar, and then select “Client Portal” on the left side of the page. From here, you’ll see the option to specify which input fields are required for clients to input when submitting a funding request. Simply click to select these.

<img class="alignnone size-medium wp-image-1056" src="https://hubtran.wpmudev.host/wp-content/uploads/2020/03/image001-300x168.jpg" alt="" width="300" height="168" srcset="https://hubtran.wpmudev.host/wp-content/uploads/2020/03/image001-300x168.jpg 300w, https://hubtran.wpmudev.host/wp-content/uploads/2020/03/image001.jpg 518w" sizes="(max-width: 300px) 100vw, 300px" /> 

&nbsp;

**2. Automating client specification of funding methods**

&nbsp;

This adds a new feature, allowing users to directly pass funding amounts into FactorSoft and eliminating an extra step.

&nbsp;

**Note that this functionality Requires FactorSoft version 3.16 SP2 or later, and additional setup from FactorSoft support.** If you’d like to set this up, please email <support@hubtran.com> and LaDona Caluory at Factorsoft, <LCaluory@jackhenry.com>.

&nbsp;

With this feature, clients using the portal can pick from their own list of bank accounts or funding methods. Once entered into the HubTran portal, these selections are automatically passed directly into FactorSoft, eliminating the need for manual entry.

&nbsp;

As with the previous feature, this can be enabled at edited in Settings >> Client Portal and will display the following prompt when your clients submit invoices:

<img class="alignnone size-medium wp-image-1057" src="https://hubtran.wpmudev.host/wp-content/uploads/2020/03/image002-244x300.jpg" alt="" width="244" height="300" srcset="https://hubtran.wpmudev.host/wp-content/uploads/2020/03/image002-244x300.jpg 244w, https://hubtran.wpmudev.host/wp-content/uploads/2020/03/image002.jpg 267w" sizes="(max-width: 244px) 100vw, 244px" /> 

&nbsp;
